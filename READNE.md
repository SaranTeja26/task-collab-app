# Task Collaboration App

## 🛠️ Setup

1. Clone repo
2. Import `db/task_collab.sql` into your MySQL database
3. Update `config/db.php` with your DB credentials
4. Serve via localhost/XAMPP
5. Run `npx tailwindcss ...` to build CSS
